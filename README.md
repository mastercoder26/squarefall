# Square Fall

A small browser game built with vanilla JavaScript and the Canvas API. No frameworks, no build step, just open `index.html` and play.

## How it works

A green paddle bounces back and forth automatically at the bottom of the screen. Red squares of varying sizes drop from the top at a steady pace. Press `O` to flip the paddle's direction and intercept them. Miss three squares and it's over.

## Controls

| Key | Action |
|-----|--------|
| O   | Reverse the paddle's direction |

## Running it

Clone the repo and open `index.html` in any modern browser. There are no dependencies to install.
